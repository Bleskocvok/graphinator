
# graphinator

## Installation

```sh
# Prerequisites:
sudo apt update
sudo apt install xfce4-dev-tools \
    libxfce4ui-2-dev \
    libxfce4panel-2.0-dev \
    imagemagick

# Install:
git clone https://github.com/Bleskocvok/graphinator.git

cd graphinator
sh ./configure
make
sudo bash script/install
```
